# Packages Installed

1. Frontend packages
   1. npm install react-router-dom socket.io-client react-emoji query-string
   2. npm install -D tailwindcss postcss autoprefixer
   3. npx tailwindcss init -p
2. Backend packages
   1. npm install cors nodemon express socket.io
